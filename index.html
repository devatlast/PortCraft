About PortCraft

PortCraft is an innovative interactive web application that transforms how professionals build their digital presence. It combines a persuasive user journey with a professional portfolio builder featuring a personalized signature sketch pad. The strategic Rock-Paper-Scissors game mechanic gamifies the experience, encouraging users to actively commit to creating a polished, personalized portfolio that authentically represents their unique skills and professional identity.
